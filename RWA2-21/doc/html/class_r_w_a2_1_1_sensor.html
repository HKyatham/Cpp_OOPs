void RWA2::Sensor::read_data ( unsigned int  duration)

This method takes duration as input and then generates 50 random values for the sensor data after sleeping for the set duration of time, as we don't have any physical sensors.

The sensor data values are in a range of 0 to 30 as the max distance measure was 30m.
